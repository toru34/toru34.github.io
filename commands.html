Useful commands

tar

  1. $ tar -xzvf file.tar.gz # Extract archive file
  2. $ tar -zcvf file.tar.gz /path/to/dir # Create archive file

Docker

  1. $ docker run -it \
        --mount type=bind,source=<source-dir>,target=<target-dir> \
        <docker-image> /bin/bash # Start docker container interactively with a mount
